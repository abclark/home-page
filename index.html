Communication Systems from Scratch
Built the internet protocol stack from first principles:

  • TCP Chat — secure chat: TCP → TLS → end-to-end encryption (RSA + AES-GCM)
  • TCP/IP Stack — user-space networking: TUN → IPv4 → ICMP → UDP → TCP
  • QUIC — from scratch: UDP → reliability → Diffie-Hellman → 0-RTT → connection migration
  • Protocol Buffers — wire format: varints → tags → encode/decode
  • HTTP/3 — on top of QUIC: frames → headers → working client/server
  • BGP — finite state machine + path selection
  • BBR — Google's congestion control algorithm
  • Audio Modem — FSK modulation, FFT, CRC — TCP packets transmitted as sound through air
